- Microservice
- Cloud Load Balancing
- Cloud Run
- gRPC
- gRPC-Gateway
- Authenticating service-to-service

- Cloud SQL For PostgreSQL
- Cloud Trace With Opentelemetry
- Firebase Authentication
- Logging (go-logr)
- Terraform
